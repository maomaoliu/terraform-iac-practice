# WEB Application API


## 环境和版本

### 基础设施

- JDK 17
- Gradle 7.4
- Docker

### 资源库

#### 工具库

- lombok
- mapstruct
- Spring Boot 3.0
- springdoc

#### 测试工具库

- junit 5
- spring boot test
- archunit
- assertj
- mockito

#### 代码校验

- Editorconfig - Google style
- PMD
- Jacoco
- Spotless


## 运行

### IDE

通过程序入口 `BootstrapApplication.java` 可直接运行


## 提交

- 提交前执行 `./script/pre-push.sh`
